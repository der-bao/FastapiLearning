# asyncio

# 一、基本概念
1. **协程** Coroutine
- 一种特殊函数，可以在执行中暂停，并在稍后恢复执行。
- 协程通过 async def 关键字定义，并通过 await 关键字暂停执行，等待异步操作完成。

2. **事件循环** Event Loop
- asyncio 的核心调度器，负责：
    - 管理所有协程 / 任务
    - 监听 I/O 事件（网络、文件、定时器）
    - 按优先级调度可执行任务
- 单线程循环：不断地检查是否有任务需要执行，并在任务完成后调用相应的回调函数。
- 入口：asyncio.run(main())（Python 3.7+），自动创建 / 关闭循环。

3. **任务** Task
- 协程的封装，是事件循环直接调度的对象。
- 用 asyncio.create_task() 创建，将协程立即加入循环并发执行。
- 支持：获取结果、取消、查看状态（done()/result()）。

4. **Future**
- 底层抽象，代表异步操作的未来结果（占位符）。
- Task 是 Future 的子类，日常开发一般不用直接操作 Future。
